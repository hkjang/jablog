# SEO 최적화 가이드

## 개요

JaBlog는 콘텐츠의 SEO(검색 엔진 최적화) 품질을 자동으로 분석하고 점수를 산출합니다. 개선 포인트를 확인하고 권장 조치를 따라 블로그의 검색 순위를 향상시킬 수 있습니다.

---

## SEO 점수 시스템

### 점수 산정 기준

| 항목            | 배점  | 감점 기준             | 설명                   |
| --------------- | ----- | --------------------- | ---------------------- |
| 콘텐츠 길이     | -20점 | 300단어 미만          | 최소 300단어 이상 권장 |
| H1 제목         | -20점 | H1 태그 없음          | 메인 제목(#) 필수      |
| 메타 설명       | -15점 | 메타 설명 없거나 짧음 | 150-160자 권장         |
| 이미지 Alt 태그 | -10점 | Alt 태그 누락         | 모든 이미지에 필수     |
| 내부 링크       | -10점 | 내부 링크 없음        | 관련 콘텐츠 연결       |
| 외부 링크       | -10점 | 외부 링크 없음        | 신뢰성 있는 출처 연결  |
| 키워드 밀도     | -10점 | 너무 낮거나 높음      | 1-3% 권장              |
| 제목 키워드     | -5점  | 제목에 키워드 없음    | 주요 키워드 포함       |

**총점: 100점 기준**

## 개선 포인트 상세 설명

### 1. 콘텐츠 길이

| 항목      | 현재 기준          | 권장 사항        |
| --------- | ------------------ | ---------------- |
| 최소 길이 | 300단어            | 500~1500단어     |
| 최적 길이 | 1000-2000단어      | 주제에 따라 조절 |
| 확인 방법 | SEO 분석 자동 확인 | -                |

**권장 행동:**

1. 본문 내용을 충실히 작성합니다.
2. 예시, 사례, 설명을 추가합니다.
3. 관련 정보를 섹션별로 정리합니다.

---

### 2. H1 제목 (메인 제목)

| 항목      | 기준     | 권장 사항        |
| --------- | -------- | ---------------- |
| 필수 여부 | 필수     | 1개의 H1 제목    |
| 길이      | 50-60자  | 핵심 키워드 포함 |
| 형식      | `# 제목` | 마크다운 H1      |

**권장 행동:**

1. 글 첫 줄에 `# 제목` 형태로 작성합니다.
2. 주요 키워드를 제목에 포함합니다.
3. 제목은 명확하고 간결하게 작성합니다.

---

### 3. 메타 설명

| 항목   | 기준             | 권장 사항           |
| ------ | ---------------- | ------------------- |
| 길이   | 150-160자        | 검색 결과에 표시    |
| 내용   | 핵심 내용 요약   | 클릭 유도 문구 포함 |
| 키워드 | 주요 키워드 포함 | 자연스럽게 삽입     |

**권장 행동:**

1. 콘텐츠의 핵심 내용을 1-2문장으로 요약합니다.
2. 주요 키워드를 자연스럽게 포함합니다.
3. 독자가 클릭하고 싶게 만드는 문구를 작성합니다.

---

### 4. 이미지 최적화

| 항목     | 기준           | 권장 사항               |
| -------- | -------------- | ----------------------- |
| Alt 태그 | 필수           | 이미지 설명 텍스트      |
| 파일명   | 의미 있는 이름 | keyword-description.jpg |
| 크기     | 최적화         | 100KB 이하 권장         |

**권장 행동:**

1. 모든 이미지에 Alt 텍스트를 추가합니다: `![이미지 설명](url)`
2. 파일명에 키워드를 포함합니다.
3. 이미지 크기를 최적화하여 로딩 속도를 개선합니다.

---

### 5. 링크 구조

| 링크 유형   | 기준     | 권장 사항        |
| ----------- | -------- | ---------------- |
| 내부 링크   | 최소 1개 | 관련 콘텐츠 연결 |
| 외부 링크   | 최소 1개 | 신뢰성 있는 출처 |
| 앵커 텍스트 | 설명적   | 키워드 포함      |

**권장 행동:**

1. 관련된 다른 블로그 글 링크를 추가합니다.
2. 참고한 자료의 출처 링크를 포함합니다.
3. "여기 클릭" 대신 설명적인 링크 텍스트를 사용합니다.

---

### 6. 키워드 밀도

| 항목        | 기준    | 권장 사항          |
| ----------- | ------- | ------------------ |
| 적정 밀도   | 1-3%    | 자연스러운 삽입    |
| 과도한 밀도 | 3% 초과 | 키워드 스터핑 주의 |
| 너무 낮음   | 1% 미만 | 키워드 추가 필요   |

**계산 방법:**

```
키워드 밀도 = (키워드 등장 횟수 / 전체 단어 수) × 100
```

## 자동 SEO 최적화 기능

### 설정 방법

1. **설정** → **AI 설정** 탭으로 이동합니다.
2. **자동 SEO 최적화** 옵션을 활성화합니다.
3. AI가 콘텐츠 작성 시 SEO 요소를 자동으로 포함합니다.

### 자동 최적화 항목

| 항목        | 자동화 여부 | 설명                   |
| ----------- | ----------- | ---------------------- |
| 제목 최적화 | ✅          | 키워드 포함 제목 생성  |
| 메타 설명   | ✅          | 콘텐츠 기반 자동 생성  |
| 제목 구조   | ✅          | H1, H2, H3 자동 구성   |
| 키워드 밀도 | ✅          | 자연스러운 키워드 분배 |
| 이미지 Alt  | ❌          | 수동 입력 필요         |

## 팁 & 권장사항

1. **독자 우선**: SEO보다 독자에게 유용한 콘텐츠가 우선입니다.
2. **자연스러움**: 키워드를 억지로 넣지 말고 자연스럽게 포함하세요.
3. **구조화**: 제목, 소제목, 단락으로 글을 구조화하세요.
4. **정기 검토**: 기존 콘텐츠도 SEO 점수를 확인하고 개선하세요.
5. **모바일 최적화**: 모바일에서도 읽기 좋게 작성하세요.
