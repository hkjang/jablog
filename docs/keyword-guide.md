# 키워드·주제 선정 가이드

## 개요

JaBlog는 Google Trends API를 활용하여 트렌드 기반 키워드를 분석하고 블로그 주제를 추천합니다. 초보자도 쉽게 활용할 수 있도록 키워드 점수, 트렌드 변화, 추천 로직을 설명합니다.

---

## 키워드 분석 기능

### 기능 설명

| 기능        | 설명                           | 데이터 출처   |
| ----------- | ------------------------------ | ------------- |
| 연관 키워드 | 입력 주제와 관련된 검색어 추출 | Google Trends |
| 검색량 분석 | 키워드별 상대적 검색량 표시    | Google Trends |
| 트렌드 분석 | 키워드 검색량 변화 추이        | Google Trends |
| 점수 산출   | 키워드 경쟁력/활용도 점수화    | 자체 알고리즘 |

---

### 키워드 점수 이해

| 점수 요소            | 설명                         | 가중치 |
| -------------------- | ---------------------------- | ------ |
| 검색량 (Volume)      | 월간 예상 검색 횟수          | 40%    |
| 경쟁도 (Competition) | 해당 키워드 콘텐츠 경쟁 정도 | 30%    |
| 트렌드 (Trend)       | 최근 검색량 변화 방향        | 30%    |

**점수 해석:**

| 점수 구간 | 의미      | 권장 조치          |
| --------- | --------- | ------------------ |
| 80-100    | 매우 우수 | 즉시 콘텐츠 작성   |
| 60-79     | 우수      | 콘텐츠 작성 권장   |
| 40-59     | 보통      | 다른 키워드와 조합 |
| 0-39      | 낮음      | 활용 재고려        |

---

### 트렌드 변화 해석

| 트렌드 표시          | 의미        | 설명                    | 권장               |
| -------------------- | ----------- | ----------------------- | ------------------ |
| 🔥 급상승 (Breakout) | 폭발적 증가 | 최근 급격히 검색량 증가 | 빠른 대응 필요     |
| ↗️ 상승              | 점진적 증가 | 검색량이 꾸준히 증가    | 콘텐츠 제작 적합   |
| → 안정               | 변화 없음   | 검색량 유지 중          | 장기 콘텐츠에 적합 |
| ↘️ 하락              | 감소 추세   | 검색량이 줄어드는 중    | 신중한 접근        |

---

## 단계별 사용 방법

### 1단계: 주제 등록

1. **콘텐츠** 또는 **파이프라인** 메뉴로 이동합니다.
2. **주제 관리** 섹션을 찾습니다.
3. 분석하고 싶은 주제명을 입력합니다.
4. **등록** 버튼을 클릭합니다.

**주제 입력 예시:**

| 입력 주제     | 분석 결과 키워드 예시                  |
| ------------- | -------------------------------------- |
| 블로그 마케팅 | SEO 최적화, 콘텐츠 마케팅, 키워드 분석 |
| AI 기술       | ChatGPT, 생성형 AI, 머신러닝           |
| 건강 관리     | 운동 루틴, 다이어트 식단, 수면 개선    |

---

### 2단계: 키워드 분석 실행

1. 등록된 주제 옆 **키워드 분석** 버튼을 클릭합니다.
2. 시스템이 Google Trends API를 호출합니다.
3. 분석 결과가 테이블 형태로 표시됩니다.

**분석 결과 테이블:**

| 키워드           | 검색량 | 경쟁도 | 트렌드    | 점수 |
| ---------------- | ------ | ------ | --------- | ---- |
| SEO 최적화       | 12,000 | 중간   | ↗️ 상승   | 85   |
| 콘텐츠 마케팅    | 8,500  | 낮음   | → 안정    | 72   |
| 블로그 수익화    | 15,000 | 높음   | ↗️ 상승   | 68   |
| 키워드 분석 도구 | 3,200  | 낮음   | 🔥 급상승 | 78   |

---

### 3단계: 키워드 선택 및 활용

1. 분석 결과에서 점수가 높은 키워드를 선택합니다.
2. 선택한 키워드를 클릭하면 콘텐츠 작성 화면으로 이동합니다.
3. 키워드가 태그로 자동 추가됩니다.

---

## 주제 추천 기능

### AI 기반 추천

대시보드의 **AI 추천** 섹션에서 자동 추천 주제를 확인할 수 있습니다.

| 추천 유형   | 설명                         | 근거                 |
| ----------- | ---------------------------- | -------------------- |
| 트렌드 기반 | 최근 급상승 키워드 기반 주제 | Google Trends 데이터 |
| 성과 기반   | 기존 고성과 콘텐츠 관련 주제 | 블로그 분석 데이터   |
| 계절 기반   | 시즌/이벤트 관련 주제        | 캘린더 분석          |

## 초보자 예시

### 예시 1: 기술 블로그 운영자

**상황:** "AI" 관련 콘텐츠를 작성하고 싶음

1. 주제로 "인공지능" 등록
2. 키워드 분석 실행
3. 결과 확인:

| 키워드         | 추천도 | 이유                     |
| -------------- | ------ | ------------------------ |
| ChatGPT 활용법 | ⭐⭐⭐ | 검색량 높음, 트렌드 상승 |
| AI 도구 추천   | ⭐⭐⭐ | 경쟁도 낮음, 트렌드 안정 |
| 머신러닝 기초  | ⭐⭐   | 검색량 중간, 전문성 필요 |

4. "ChatGPT 활용법"으로 콘텐츠 작성 시작

**상황:** "다이어트" 관련 글을 쓰고 싶음

1. 주제로 "다이어트" 등록

3. 결과 확인:

## 키워드 활용 팁

| 팁            | 설명                                  |
| ------------- | ------------------------------------- |
| 롱테일 키워드 | 2-3개 단어 조합으로 경쟁도 낮춤       |
| 시즌 키워드   | 명절, 계절 관련 키워드 미리 선점      |
| 질문형 키워드 | "~하는 방법", "~란?" 형태 활용        |
| 지역 키워드   | "서울 맛집", "강남 카페" 등 지역 조합 |

## 자주 묻는 질문

### Q1. 키워드 분석 결과가 없습니다.

Google Trends에서 데이터를 가져올 수 없는 경우입니다. 주제명을 더 일반적인 용어로 변경해보세요.

### Q2. 검색량이 0으로 표시됩니다.

검색량이 매우 적거나 새로운 키워드일 수 있습니다. 급상승(Breakout) 표시가 있다면 신규 트렌드입니다.

### Q3. 점수가 높은데 경쟁이 많습니다.

경쟁이 많은 키워드는 차별화된 관점이나 롱테일 키워드로 접근하세요.
