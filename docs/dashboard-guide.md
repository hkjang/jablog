# 대시보드 화면 가이드

## 개요

JaBlog 대시보드는 블로그 성과를 한눈에 파악하고 효율적으로 콘텐츠를 관리할 수 있도록 설계되었습니다. 사용자 편의성을 최우선으로 핵심 지표, AI 추천, 예정 발행, 콘텐츠 현황을 직관적으로 표시합니다.

---

## 화면 구성 요소

### 1. 헤더 영역

| 구성 요소   | 설명                         | 기능             |
| ----------- | ---------------------------- | ---------------- |
| 페이지 제목 | "대시보드"                   | 현재 페이지 식별 |
| 서브타이틀  | "블로그 성과 및 콘텐츠 관리" | 페이지 목적 설명 |

---

### 2. 핵심 지표 카드 (Metrics Grid)

상단에 4개의 핵심 지표 카드가 그리드 형태로 배치되어 있습니다.

| 카드명        | 아이콘        | 표시 데이터  | 트렌드 표시   | 기능               |
| ------------- | ------------- | ------------ | ------------- | ------------------ |
| 총 조회수     | 👁️ (Eye)      | 전체 조회 수 | ✅ 상승/하락% | 조회수 추이 확인   |
| 총 클릭수     | 🖱️ (Click)    | 전체 클릭 수 | ✅ 상승/하락% | 클릭수 추이 확인   |
| 평균 클릭률   | 📊 (BarChart) | 클릭률 %     | ❌            | CTR 성과 확인      |
| 평균 체류시간 | ⏱️ (Clock)    | 초 단위      | ❌            | 사용자 참여도 파악 |

**트렌드 아이콘 설명:**

- 🔼 녹색 화살표: 전월 대비 상승
- 🔽 빨간색 화살표: 전월 대비 하락
- `-`: 변화 없음

---

### 3. 알림 섹션 (Alerts)

| 구성 요소   | 설명                    | 기능                 |
| ----------- | ----------------------- | -------------------- |
| 섹션 제목   | "⚠️ 알림"               | 알림 영역 표시       |
| 알림 리스트 | 알림 카드 목록          | 중요 알림 확인       |
| 알림 타입   | info, warning, error 등 | 색상으로 중요도 구분 |
| 제목/메시지 | 알림 내용               | 상세 정보 표시       |

**알림 유형:**

| 유형    | 색상   | 용도           |
| ------- | ------ | -------------- |
| info    | 파란색 | 일반 정보 알림 |
| warning | 노란색 | 주의 필요 알림 |
| error   | 빨간색 | 오류/긴급 알림 |
| success | 녹색   | 성공 알림      |

---

### 4. AI 추천 섹션 (Recommendations)

| 구성 요소   | 설명         | 기능                       |
| ----------- | ------------ | -------------------------- |
| 섹션 제목   | "💡 AI 추천" | AI 분석 기반 추천          |
| 개선 제안   | 텍스트 목록  | 콘텐츠 개선 방향 제시      |
| 추천 키워드 | 태그 형태    | 검색량과 함께 키워드 표시  |
| 키워드 볼륨 | 숫자 배지    | 예상 검색량 (K=천, M=백만) |

**추천 키워드 태그 예시:**

| 키워드      | 검색량 | 경쟁도 | 트렌드 |
| ----------- | ------ | ------ | ------ |
| AI 블로그   | 10.5K  | 낮음   | 상승   |
| SEO 최적화  | 8.2K   | 중간   | 안정   |
| 자동화 도구 | 5.1K   | 낮음   | 상승   |

---

### 5. 예정된 발행 섹션 (Upcoming Scheduled)

| 구성 요소   | 설명              | 기능             |
| ----------- | ----------------- | ---------------- |
| 섹션 제목   | "📅 예정된 발행"  | 예약 발행 일정   |
| 예약 목록   | 발행 대기 콘텐츠  | 예정 콘텐츠 확인 |
| 콘텐츠 제목 | 발행할 글 제목    | 어떤 글인지 식별 |
| 플랫폼      | WordPress/Tistory | 발행 대상 플랫폼 |
| 예약 시간   | 월, 일, 시:분     | 발행 예정 시각   |

**빈 상태 메시지:** "예정된 발행이 없습니다."

---

### 6. 콘텐츠 현황 섹션 (Pipeline Stats)

| 구성 요소     | 설명          | 기능                 |
| ------------- | ------------- | -------------------- |
| 섹션 제목     | "콘텐츠 현황" | 상태별 콘텐츠 집계   |
| 상태별 카운트 | 숫자 + 상태명 | 워크플로우 현황 파악 |

**콘텐츠 상태 표시:**

| 상태      | 설명   | 표시 형식      |
| --------- | ------ | -------------- |
| DRAFT     | 초안   | 5 / DRAFT      |
| REVIEW    | 검토중 | 3 / REVIEW     |
| APPROVED  | 승인됨 | 8 / APPROVED   |
| SCHEDULED | 예약됨 | 2 / SCHEDULED  |
| PUBLISHED | 발행됨 | 15 / PUBLISHED |

## 사용 팁

1. **대시보드 새로고침**: 페이지 로드 시 자동으로 최신 데이터를 가져옵니다.
2. **트렌드 확인**: 조회수/클릭수 카드의 트렌드 화살표로 성과 변화를 빠르게 파악하세요.
3. **AI 추천 활용**: 추천 키워드를 클릭하면 해당 키워드로 콘텐츠 작성을 시작할 수 있습니다.
4. **예약 관리**: 예정된 발행 섹션에서 일정을 확인하고 캘린더 메뉴로 이동하여 관리하세요.
5. **오류 알림 확인**: 빨간색 알림은 즉시 확인하고 조치가 필요합니다.
