# JaBlog 사용자 가이드

## 1. 서비스 개요

JaBlog는 블로그 콘텐츠 자동화 플랫폼입니다. AI 기반 콘텐츠 생성, 다중 플랫폼(WordPress, Tistory) 동시 발행, 예약 발행, 키워드 분석 및 SEO 최적화 기능을 제공합니다. 트렌드 기반 주제 추천과 성과 분석 대시보드를 통해 블로그 운영 효율을 극대화할 수 있습니다.

## 4. 단계별 사용 방법

### 4.1 시작하기

1. 브라우저에서 JaBlog 대시보드에 접속합니다.
2. 이메일과 비밀번호로 로그인합니다.
3. 대시보드 화면에서 현재 성과 지표를 확인합니다.

### 4.2 플랫폼 연동 설정

1. 좌측 메뉴에서 **설정**을 클릭합니다.
2. **플랫폼 연동** 탭을 선택합니다.
3. **새 플랫폼 연동 추가** 버튼을 클릭합니다.
4. 플랫폼(WordPress 또는 Tistory)을 선택합니다.
5. 필수 정보를 입력합니다:
   - **WordPress**: 사이트 URL, 사용자명, 앱 비밀번호
   - **Tistory**: 블로그명, Access Token
6. **추가** 버튼을 클릭합니다.
7. **테스트** 버튼으로 연결 상태를 확인합니다.

### 4.3 콘텐츠 생성

1. **콘텐츠** 메뉴로 이동합니다.
2. **새 콘텐츠 작성** 버튼을 클릭합니다.
3. 제목, 본문, 카테고리, 태그를 입력합니다.
4. AI 자동 생성을 원하면 **AI 생성** 체크박스를 활성화합니다.
5. **저장** 버튼을 클릭합니다.

### 4.4 예약 발행 설정

1. **예약 캘린더** 메뉴로 이동합니다.
2. 원하는 날짜를 클릭합니다.
3. 발행할 콘텐츠를 선택합니다.
4. 발행 시간과 플랫폼을 지정합니다.
5. **예약 등록** 버튼을 클릭합니다.

### 4.5 성과 분석 확인

1. **성과 분석** 메뉴로 이동합니다.
2. 기간 필터로 조회 기간을 선택합니다.
3. 조회수, 클릭률, 수익 등 지표를 확인합니다.
4. 플랫폼별/콘텐츠별 상세 분석을 확인합니다.

## 5. 자주 묻는 질문 (FAQ)

### Q1. WordPress 앱 비밀번호는 어디서 생성하나요?

WordPress 관리자 페이지 → 사용자 → 프로필 → 애플리케이션 비밀번호 섹션에서 새 비밀번호를 생성할 수 있습니다.

### Q2. Tistory Access Token은 어떻게 얻나요?

Tistory Open API에서 앱을 등록한 후 OAuth 인증을 통해 Access Token을 발급받을 수 있습니다.

### Q3. AI 콘텐츠 생성이 되지 않습니다.

설정 → AI 설정에서 API Key가 올바르게 입력되었는지 확인하세요. Ollama 사용 시 로컬 서버가 실행 중인지 확인하세요.

### Q4. 예약 발행이 실행되지 않습니다.

백엔드 서버가 계속 실행 중인지 확인하세요. 예약 시간이 지났는데 발행되지 않았다면 API 모니터링에서 오류를 확인하세요.

### Q5. 플랫폼 연결 테스트가 실패합니다.

인증 정보(API Key, 비밀번호 등)가 올바른지 확인하세요. WordPress의 경우 REST API가 활성화되어 있어야 합니다.

## 6. 오류 해결

| 오류 유형   | 증상             | 원인               | 해결 방법                       | 예방 팁              |
| ----------- | ---------------- | ------------------ | ------------------------------- | -------------------- |
| 연결 오류   | 플랫폼 연결 실패 | 잘못된 인증 정보   | 인증 정보 재확인 후 재입력      | 정기적으로 토큰 갱신 |
| 발행 오류   | 포스팅 실패      | 플랫폼 API 오류    | API 모니터링에서 상세 오류 확인 | 발행 전 연결 테스트  |
| AI 오류     | 콘텐츠 생성 실패 | API 키 만료/잘못됨 | AI 설정에서 API 키 재설정       | API 잔액 모니터링    |
| 로그인 오류 | 로그인 불가      | 비밀번호 불일치    | 비밀번호 재설정                 | 비밀번호 관리자 사용 |
| 데이터 오류 | 데이터 로딩 실패 | 서버 연결 문제     | 페이지 새로고침, 서버 상태 확인 | 브라우저 캐시 정리   |
| SEO 오류    | SEO 점수 미표시  | 콘텐츠 분석 실패   | 본문 길이 확인 (최소 300단어)   | 충분한 콘텐츠 작성   |
